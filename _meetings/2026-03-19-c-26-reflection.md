agenda: >-
  ### Introduction to reflections in C++26


  Vasil Pashov will talk about one of the biggest innovations in the language in the last ~20 years – static reflection. It allows us not only to extract information that the compiler knows about the code (names of variables, public/private members, function arguments, template parameters, etc.), but also to use that information to generate new code. This makes tasks such as:


  * automatic UI generation

  * serialization (JSON, XML, etc.)

  * transformations like structure of arrays ↔ arrays of structures


  trivial
